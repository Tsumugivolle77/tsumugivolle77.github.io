Map similar obj into the same bucket.

Distance Measures

Jaccard distance for measurement of dist. between 2 sets.

3 Steps for similar docs

  1. Shingling: docs. -> sets (not like prob. dist.)
  2. Min-hashing: (maybe large) sets -> short signatures, preserving similarity info.
  3. LSH
  4. Candidate pairs

k-Shingles

A sequence of k tokens that appear in the doc.

e.g. k=2, D=abcab
2-Shingle S(D)={ab, bc, ca, ab} (multiset, allows elems to appear twice or more)

Note: pick k large enough. Like k=5 ok for short doc., k=10 better for large doc.

Minhashing

Encode k-Shingle as bitset.

-> Intersection as AND, Union as OR.

Sets -> Bool matrices

rows = elems (shingles)
cols = sets (docs.)
usually sparse

Our goal: sim. of cols == sim. of signatures
h: C -> signature of C
sim(C1, C2) high -> h(C1) == h(C2),
sim(C1, C2) low -> h(C1) != h(C2).
